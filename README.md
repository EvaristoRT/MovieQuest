## 📖 About

MovieQuest is a modern React application designed to make movie discovery more engaging, interactive, and enjoyable.

Unlike traditional movie catalogs, MovieQuest focuses on solving a common problem:

> **"I want to watch a movie... but I don't know which one."**

Using data provided by **The Movie Database (TMDB)** API, the application combines powerful search capabilities with original features such as personalized recommendations, custom collections, movie challenges, watch history, and viewing statistics.

MovieQuest is being developed as a portfolio project to demonstrate modern Front-End development practices, including reusable components, client-side routing, global state management, asynchronous JavaScript, responsive layouts, and API integration.
